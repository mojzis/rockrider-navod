# Grandma's E-Bike Guide — GitHub Pages Project

## What is this?

A friendly, large-text interactive guide for the **Rockrider E-ACTV 500** electric bike, designed for an elderly user who needs a simple always-available reference on their phone or tablet.

## How to use this with Claude Code

1. Open your terminal
2. Run: `claude` (to start Claude Code)
3. Paste the contents of `CLAUDE_CODE_PROMPT.md` as your first message
4. Claude Code will generate `index.html`

## How to deploy to GitHub Pages

1. Create a new repository on GitHub (e.g. `grandma-bike-guide`)
2. Copy `index.html` into the repo root
3. Go to **Settings → Pages**
4. Under "Source", select **Deploy from a branch**
5. Choose branch: `main`, folder: `/ (root)`
6. Click Save
7. Your site will be live at: `https://YOUR_USERNAME.github.io/grandma-bike-guide/`

Share that link with Grandma — she can bookmark it on her phone! 📱

## Contents covered

- 🔋 Turning the bike on/off + inserting/removing battery
- 💡 Turning lights on and off
- ⚡ Choosing assistance modes (0–4)
- 🚶 Walk mode (push the bike uphill)
- 🔌 Charging the battery
- ⚠️ Troubleshooting common problems
